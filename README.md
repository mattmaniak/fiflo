# Fiflo - the simple terminal-based text editor.

## Description
Nano-inspired. Currently unsafe && unusable because WIP.

## Compilation
This will run src/configure.sh to check your system and compile Fiflo.
```
make
```

## Installation
```
sudo make install
```

## Uninstallation
```
sudo make uninstall
```
